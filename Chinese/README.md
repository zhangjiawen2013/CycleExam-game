# 轮回答卷

## 项目描述
**轮回答卷** 是一款基于 C++ 标准库开发的控制台文本选择冒险游戏。玩家将在中考前后做出不同决策，触发多分支剧情与不同结局。

当前版本：26w11a pre2

## 核心特性
- 纯控制台交互
- 多结局设计
- 自动日志输出
- 输入验证机制
- 简易功能菜单

## 运行要求
- 操作系统：Windows（暂不兼容 Linux/macOS）
- 编译环境：支持 C++17 及以上的编译器（如 MinGW-g++、MSVC）
- 依赖：仅使用 C++ 标准库，无第三方依赖

## 编译步骤
将 `main.cpp` 与 `function_1.h` 放在同一目录，打开终端执行：
```bash
# MinGW-g++ 编译（推荐）
g++ main.cpp -std=c++17 -o main.exe

# MSVC 编译（可选）
cl main.cpp /Fe:main.exe
